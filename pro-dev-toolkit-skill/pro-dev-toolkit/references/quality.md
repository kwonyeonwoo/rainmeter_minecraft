# Quality Assurance & DevOps Playbook

## 🧪 Testing with Jest
- **AAA Pattern:** Arrange, Act, Assert.
- **Async Tests:** Use `waitFor` and `findBy` for React Testing Library.
- **Coverage:** Aim for 80%+ coverage on business logic (utils and services).

## 🐳 Infrastructure
- **Docker:** Multi-stage builds for Next.js to minimize image size.
- **CI/CD:** GitHub Actions for automated linting and PR validation.
