# 타임 키퍼 (월간 80시간 근무 스케줄러)

본 프로젝트는 월 최대 80시간을 기준으로 근무 일정을 관리하고 한눈에 확인할 수 있는 웹 기반 스마트 스케줄러입니다.

## ✨ 전체 기능 및 특징 안내

### 1. 스마트 누적 근무 설정 및 80시간 자동 마감 (캡핑)
- **제한된 근무 시간 관리**: 해당 월의 총 누적 근무 시간이 80시간에 도달하면 이후 일정은 자동으로 차단(마감 처리)되어 한도 초과를 방어합니다.
- **실시간 요약 대시보드**: 누적 근무시간과 달성률을 프로그레스 게이지 바 형태로 시각화하여 이번 달에 얼마나 일했는지 한눈에 파악할 수 있으며, 월간 출근(근무) 일수를 함께 보여줍니다.

### 2. 직관적인 다이렉트 캘린더 예외 관리 (모달 전용)
- **달력 직접 클릭 연동**: 달력의 특정 날짜 뱃지(박스)를 마우스로 클릭하면, 해당 날짜만을 위한 상세 설정 팝업(모달)창이 즉시 표출됩니다. (이전 사이드바의 입력 폼을 완전히 모달로 일원화 대체)
- **휴무/시간 개별 변동 처리**: 열린 팝업에서 해당 날짜를 '휴무'로 지정하거나, 기존과 다른 '근무 시간'으로 덮어씁니다.
- **주간 일괄 설정 보완**: [월~일요일별 기본 시간]을 일괄 적용한 뒤, 특별하게 바뀌는 날만 달력 다이렉트 클릭으로 관리해, 복잡하고 유동적인 일정도 가장 빠르게 세팅합니다. 
- **사이드바 기록 리스트 가독성 강화 (NEW)**: 달력을 덮어쓴 예외일 내역들이 왼쪽 사이드바 패널에 일목요연하게 표시됩니다. 리스트에는 단순히 날짜뿐 아니라 **요일 정보**가 괄호로 추가되었으며, 각 항목이 카드로 분리되고 **색상 뱃지(Badge)** 형태(예: 붉은색 `[휴무 처리]`, 보라색 `[N시간 근무]`)로 적용되어 스크롤 시에도 변경 이력을 0.1초만에 파악할 수 있는 압도적인 가독성을 자랑합니다.

### 3. 직관적인 달력 시각적 뱃지 시스템
달력 내의 모든 날짜에는 상태에 따라 서로 다른 색상의 뱃지가 부착되어 정보를 명확히 구분합니다.
- 🔵 **일반 할당 (파란색)**: 사이드바 주간 기본 설정에 의해 자동 채워진 정상 근무
- 🟣 **예외 지정 (보라색)**: 모달 팝업으로 추가/단축 시간을 임의 조장하여 반영된 근무
- 🔴 **휴무 처리 (붉은색)**: 사용자가 직접 달력 모달을 통해 단일 [휴무]로 명시한 날짜
- 🟢 **마감됨 (초록색)**: 누적 시간이 80시간 누적 한계에 도달해, 남은 짜투리 한도까지만 채우고 강제 스톱된 일자

### 4. 안전한 로컬 보관 (상태 영구 보존)
- 별도의 서버 처리 없이 브라우저 자체 캐싱 기술인 **로컬 스토리지(Local Storage)** 방식을 사용하므로, 새로고침이나 브라우저 재시작을 하더라도 사용자가 지정한 일정과 클릭 변동 데이터가 안전하게 100% 보존 유지됩니다.
- 초기화 버튼(`🔄 모든 데이터 초기화`)을 눌러 언제든지 새로운 달력을 깨끗한 환경에서 시작할 수 있습니다.

### 5. 고성능 UI 최적화 렌더링 (다크 테마)
- 눈이 편한 Dark Theme 환경 하에서 투명도 조합을 95% 솔리드 패널로 설계, 무거운 블러 연산을 완전히 덜어내는 최적화를 달성하여 환경에 종속되지 않고 언제나 **가볍고 부드러운 스크롤 퍼포먼스**를 제공합니다.

### 5. 스케줄러 내보내기 (화면 캡쳐 툴)
- 💾 **[캡처 저장]** 버튼을 통해 완성된 스케줄표 화면 전체 영역을 외부의 별도 도구 없이 아주 선명한 이미지 파일(.png) 형식으로 깔끔히 자동 크롭, 다운로드 할 수 있습니다. 팀 소통에 유용합니다.

### 6. 모바일 스마트폰 네이티브 앱 설치 지원 (PWA 적용) (NEW!)
본 스케줄러는 **프로그레시브 웹 앱(PWA)** 기술이 적용되어 아이폰(iOS) 또는 안드로이드 스마트폰에 일반 어플처럼 저장할 수 있습니다!
- **어플 설치 방법**: 
  1. 본 `index.html` 파일을 모바일 인터넷(사파리, 크롬)으로 접속합니다.
  2. 공유(보내기) 또는 메뉴 버튼을 눌러 **[홈 화면에 추가]**를 클릭합니다.
  3. 스마트폰 배경화면에 '타임 키퍼' 초시계(⏱️) 아이콘의 전용 어플이 생성됩니다.
- **오프라인 네이티브 구동**: 설치된 모바일 타임키퍼를 실행하면 브라우저 주소창과 화면 테두리가 모두 사라진 완벽한 풀스크린 네이티브 앱 형태로 진입합니다. 특히 브라우저 캐시에 엔진이 내장되어 네트워크(Wi-Fi, 데이터)가 100% 끊어진 환경(지하, 비행기)에서도 스케줄 추가 및 저장이 완벽하게 작동합니다. (서비스 워커 탑재)
- **반응형 세로 모드 지원**: 스마트폰 좁은 비율에 맞춰 사이드바는 하단 스크롤로, 메인 달력은 한눈에 들어오는 가독성 높은 상단으로 최적화 배치되었습니다.
